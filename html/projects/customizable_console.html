Customizable Console

Make awesome looking cross-platform console applications

Your program, your style.

The "ConsoleColor.h" library offers cross-platform customization for console applications. It simplifies text formatting, draws shapes, and displays user-friendly messages. It saves time, aids collaboration, and enhances user experience by abstracting complex console manipulations.


Download Documentation

This library is written in

C


C is a very powerful programming language. It is used a lot on embedded systems and command line tools. Which is why we chose C to make this Customization library. It also gives you full access over memory.
